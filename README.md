# discord-rename-bot

## Setup

1. Rename example.env to .env
2. Insert your discord token in .env

### Linux

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```

### Windows

```bash
python -m venv venv
.\venv\Scripts\activate.bat
pip install -r requirements.txt
python main.py
```
